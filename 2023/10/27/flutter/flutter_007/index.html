Flutter 中 List 用法大全

前言

1、赋值与填充

  • List 的初始化方法有多种，唯一要注意的是 filled 初始化后不能直接 addAll

2、查找

  • List 在查找时注意带不带 index，带 index 则是找位置

3、插入

  • List 在插入时注意插入的位置，插入后，原 index 位置的元素会向后挪

4、删除

  • List 在删除时要注意是删除位置还是删除元素

5、替换

  • List 在替换时，要注意位置及是否跳过

6、排序

  • List 在排序时，注意是倒序还是正序

7、检查和判断

  • List 在排序时，注意是倒序还是正序

8、展开和拼接

  • List 在拼接时，注意 expand 的灵活运用

9、遍历

  • List 在遍历时，注意 forEach 一般不推荐

10、截取

  • List 的截取方法很多，注意灵活运用
